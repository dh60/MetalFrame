# MetalFrame
A video player for macOS using Metal 4 and MetalFX for high quality playback.
## Features
- Metal 4 pipeline with native rendering (no OS scaling)
- MetalFX upscaling
## Keybinds
| Key | Action |
|-----|--------|
| Right / Down | Forward 10s |
| Left / Up | Back 10s |
| i | Toggle info overlay |
| f | Toggle Fullscreen |
| Space | Play / Pause |
| Esc | Quit |
## Info Overlay
Press `i` to show the info overlay, which displays video details and provides toggles for:
- **Scaling** - Enable/disable all scaling
## Requirements
- macOS 26+
- Apple Silicon
